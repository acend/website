abstract: "Kubernetes ist ein Open Source System zur Automatisierung der Bereitstellung, Skalierung und Verwaltung von Container-Anwendungen. Es wurde von Google entwickelt und wird von der Cloud Native Computing Foundation (CNCF) unterstützt. Kubernetes ist die führende Plattform für Container Orchestrierung und wird von vielen Unternehmen weltweit eingesetzt."
