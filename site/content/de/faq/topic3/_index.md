abstract: "Cloud Native ist ein Ansatz, um Anwendungen zu entwickeln und zu betreiben. Dabei werden die Vorteile der Cloud voll ausgeschöpft. Cloud Native Anwendungen sind modular, skalierbar und hochverfügbar. Sie werden in Containern betrieben und dynamisch orchestriert. Cloud Native Technologien sind Open Source und werden von der Cloud Native Computing Foundation (CNCF) unterstützt."
